# Softversko inženjerstvo 2026. izrada projekta (web aplikacija Licitator)

Licitator je web aplikacija razvijena u okviru predmeta Softversko inženjerstvo 1. Projekat je trenutno u procesu razvoja i ima za cilj da omogući korisnicima jednostavno i sigurno učestvovanje u online aukcijama.
Registrovani korisnici mogu objavljivati predmete, učestvovati u aukcijama postavljanjem ponuda, pratiti rezultate aukcija i dobijati notifikacije o promenama. 
Administrator nadgleda aktivnosti korisnika i upravlja sadržajem kako bi platforma bila bezbedna.
